Community Gardens

See what's growing in your neighbourhood

Link to GitHub repository

Ever wondered how you neighbour always seems to grow bigger tomatoes than you? This is no longer a secret! Although in its beginning stages this webapp has the aim of letting you in on the worlds gardening secrets. Not only this, but it aims to aid you track when plants were fertilised or havested. The idea behind this app is by using a vast amount of data from its users, predictions can be made about when to plant and harvest or protect against hash weather. Recommendations can be made as to other plants and vegetables a user can try based on the climate of their locations. Will then look at expanding this idea to recommend where these can be purchased locally.

This website is in its infancy but it will allow for the opporunity to learn about using client location within an app. The aim to use only dotnet architecture for the entire project.

FishingTrip

FishingTrip - The only website for fishing

Link to GitHub repository

Try it out through an ngrok tunnel

FishingTrip is a website designed to help fishermen(and fisherwoman!) keep track of their best catches! Not only this, it automatically keeps track of the weather and other conditions that led to that PB, and will alert when similar conditions are around the corner!

With the aim to be the facebook of the fishing world, this intuitive website has a range of features for the most mad keen of us. Organise fishing trips with friend easily, this will pass on all information such as conditions and weather to everyone invited and allows for alterations to the game plan by everyone.

Using the most accurate information available, this site is a must have tool for anyone who is looking for the best day fishing!

Currently this webapp has a log in server which allows the user to save favourite fishing spots which the forecast conditions are updated constantly updated so ensure the latest are always available. The user can also search for any fishing spots all over the world. This site gathers information from forecast sites using a python webscraper, while the interaction between client and server is through dotnet.
